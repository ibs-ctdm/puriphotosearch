# Puri Photo Search — คู่มือการใช้งาน

---

## ลองนึกภาพดูครับ...

คุณเพิ่งกลับมาจากงานบวช งานตักบาตร หรืองานทอดกฐิน มีรูปถ่ายกว่า **1,000 รูป** กระจายอยู่ในโฟลเดอร์ คุณต้องการส่งรูปให้แต่ละคนที่มาร่วมงาน — **แต่ใครจะมานั่งเลือกทีละรูป?**

**Puri Photo Search ทำให้เรื่องนี้ง่ายมาก**

เพียงเลือกโฟลเดอร์รูปภาพ กดสแกน แล้วตั้งชื่อ — โปรแกรมจะ **คัดแยกรูปเป็นโฟลเดอร์ตามชื่อคน** ให้เสร็จสรรพ พร้อมส่งต่อทันที

> **ใช้เวลาไม่ถึง 5 นาที จากรูปหลายร้อยรูป → แยกเป็นโฟลเดอร์ตามคน เรียบร้อย!**

---

## สิ่งที่ทำได้

- **จดจำใบหน้าอัตโนมัติ** — ใช้ AI (InsightFace / ArcFace) ระดับเดียวกับที่ใช้ในระบบรักษาความปลอดภัย
- **สแกนแล้วตั้งชื่อทีหลัง** — ไม่ต้องลงทะเบียนก่อน ให้ระบบสแกนจัดกลุ่มใบหน้าให้ แล้วค่อยตั้งชื่อ
- **ค้นหาจากรายชื่อ** — มีรูปอ้างอิงแล้ว ค้นหาจากโฟลเดอร์ไหนก็ได้
- **คัดแยกอัตโนมัติ** — สร้างโฟลเดอร์ตามชื่อคน แล้วคัดลอกรูปให้เอง
- **จัดกลุ่มบุคคล** — สร้างกลุ่ม ลากบุคคลเข้ากลุ่ม จัดหมวดหมู่ได้ง่าย
- **ทำงานแบบ Offline** — ไม่ต้องต่อเน็ต ไม่อัปโหลดรูปไปไหน รูปของคุณอยู่ในเครื่องตลอด
- **ใช้ได้ทั้ง macOS และ Windows**

---

## การติดตั้ง

### macOS

1. เปิดไฟล์ `PuriPhotoSearch-vX.X.X-macOS.dmg`
2. ลาก **PuriPhotoSearch** ไปวางใน **Applications**
3. เปิดโปรแกรมจาก Applications ได้เลย (ไม่มีการแจ้งเตือน Unidentified Developer เพราะโปรแกรมผ่านการ Sign และ Notarize จาก Apple แล้ว)

### Windows

1. แตกไฟล์ `PuriPhotoSearch-vX.X.X-Windows.zip`
2. เปิดโฟลเดอร์ที่แตกออกมา
3. ดับเบิลคลิก `PuriPhotoSearch.exe` เพื่อเริ่มใช้งาน

เมื่อเปิดโปรแกรมขึ้นมา จะเห็น **แถบเมนูด้านบน** แบ่งเป็น 3 เมนู:

```
┌──────────────────────────────────────────────────────────────┐
│  [โลโก้]    สแกนรูปภาพ  |  รายชื่อ  |  วิธีใช้งาน    v1.8  │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│                     พื้นที่ทำงานหลัก                          │
│                (เปลี่ยนตามเมนูที่เลือก)                       │
│                                                              │
├──────────────────────────────────────────────────────────────┤
│ ● พร้อมใช้งาน       พบ 5 โฟลเดอร์      บุคคล: 10 | รูป: 500│
└──────────────────────────────────────────────────────────────┘
```

| เมนู | หน้าที่ |
|------|--------|
| **สแกนรูปภาพ** | เลือกโฟลเดอร์ สแกนใบหน้า ค้นหาบุคคล |
| **รายชื่อ** | จัดการฐานข้อมูลบุคคล สร้างกลุ่ม |
| **วิธีใช้งาน** | คำแนะนำการใช้งาน + ข้อมูลผู้พัฒนา |

**แถบสถานะด้านล่าง** จะแสดง:

- สถานะโมเดล AI (กำลังโหลด / พร้อมใช้งาน)
- จำนวนโฟลเดอร์ที่เลือก
- สถิติฐานข้อมูล (จำนวนบุคคล, รูปภาพ, ใบหน้า)

```

  ├── 📁 งานบวชน้องเอ/        ← โฟลเดอร์ย่อย = กิจกรรม

```

## เมนู "สแกนรูปภาพ"

หน้าจอแบ่งเป็น **ซ้าย** (เลือกโฟลเดอร์) และ **ขวา** (2 แท็บ)

### ฝั่งซ้าย: เลือกโฟลเดอร์

1. กดปุ่ม **"เลือก..."** เลือกโฟลเดอร์หลักที่เก็บรูปภาพ
2. โปรแกรมจะ **สแกนโฟลเดอร์ย่อย** ทั้งหมดให้อัตโนมัติ พร้อมแสดงจำนวนรูปในแต่ละโฟลเดอร์
3. **ติ๊กเลือก** โฟลเดอร์ย่อยที่ต้องการ
   - ติ๊กที่หัว = เลือก/ยกเลิกทั้งหมด
   - ปุ่ม **รีเฟรช** = สแกนโฟลเดอร์ใหม่ (กรณีเพิ่มรูปใหม่)
4. สถานะแต่ละโฟลเดอร์: **"รอ"** (ยังไม่สแกน) หรือ **จำนวนใบหน้าที่พบ** (สแกนแล้ว)

### ฝั่งขวา: 2 แท็บ

---

### แท็บ "สแกนก่อนตั้งชื่อทีหลัง" (แนะนำ!)

> **เหมาะกับ:** เริ่มใช้งานครั้งแรก หรือยังไม่ได้ลงทะเบียนบุคคลไว้

**ขั้นตอน:**

1. ติ๊กเลือกโฟลเดอร์ทางซ้าย
2. กดปุ่ม **"เริ่มสแกนใหม่"** (สีส้ม)
3. รอระบบสแกน — จะแสดงแถบความคืบหน้า
4. เมื่อสแกนเสร็จ จะแสดง **การ์ดกลุ่มใบหน้า** ที่พบ:
   - แต่ละการ์ดมีรูปตัวอย่าง + จำนวนรูปที่พบ
   - หากตรงกับบุคคลในฐานข้อมูล จะแสดงชื่อให้อัตโนมัติ
   - พิมพ์ชื่อในช่อง "ไม่ตั้งชื่อ = ข้าม" เพื่อตั้งชื่อบุคคล
5. กดปุ่ม **"ดำเนินการ"**
   - ระบบจะคัดลอกรูปเข้าโฟลเดอร์ตามชื่อที่ตั้ง
   - เพิ่มบุคคลใหม่เข้าฐานข้อมูลอัตโนมัติ (ครั้งต่อไปใช้ค้นหาได้เลย)
6. แสดง **ตารางสรุป** — ดับเบิลคลิกเพื่อเปิดโฟลเดอร์ผลลัพธ์

**ฟีเจอร์เสริม:**

- **รวมเป็นคนเดียว** — ถ้าระบบแยกคนเดียวกันออกเป็น 2 กลุ่ม ติ๊กเลือกทั้ง 2 กลุ่มแล้วกด "รวมเป็นคนเดียว"
- กด **"← เริ่มต้นใหม่"** เพื่อสแกนรอบใหม่

---

### แท็บ "ค้นหาจากรายชื่อ"

> **เหมาะกับ:** มีรายชื่อบุคคลในฐานข้อมูลแล้ว ต้องการค้นหารูปจากโฟลเดอร์ใหม่

**ขั้นตอน:**

1. ติ๊กเลือกโฟลเดอร์ทางซ้าย
2. **ติ๊กเลือกบุคคล** ที่ต้องการค้นหา (เลือกได้หลายคนพร้อมกัน)
   - ใช้ช่องค้นหาเพื่อกรองรายชื่อ
   - ปุ่ม **"เลือกทั้งหมด"** / **"ยกเลิกทั้งหมด"**
3. ปรับ **ค่าความคล้าย** ถ้าต้องการ (ค่าเริ่มต้น 0.45 เหมาะกับส่วนใหญ่)
4. กดปุ่ม **"ดำเนินการ"**
   - ถ้าโฟลเดอร์ยังไม่เคยสแกน ระบบจะสแกนให้อัตโนมัติก่อนค้นหา
5. ผลลัพธ์จะแสดง **แยกตามบุคคล** — แต่ละคนจะมี:
   - รูปที่พบ (แสดงเป็น Grid พร้อมค่าความคล้าย)
   - ปุ่มเปิดโฟลเดอร์ผลลัพธ์ (คลิกเพื่อเปิดใน Finder/Explorer)
   - คลิกที่รูปเพื่อเปิดดูรูปเต็ม
6. กด **"← เริ่มต้นใหม่"** เพื่อกลับไปเลือกบุคคลใหม่

### ค่าความคล้าย

| ค่า | ความหมาย |
|-----|----------|
| **สูง (0.60-0.90)** | เข้มงวดมาก ได้แต่รูปที่ตรงมากๆ อาจพลาดบางรูป |
| **กลาง (0.40-0.60)** | สมดุลดี **(แนะนำ: 0.45)** |
| **ต่ำ (0.20-0.40)** | หลวม ได้รูปเยอะ แต่อาจมีรูปผิดคนปน |

## เมนู "รายชื่อ"

> จัดการฐานข้อมูลบุคคล สร้างกลุ่ม และจัดหมวดหมู่

หน้าจอแบ่งเป็น **ซ้าย** (กลุ่ม) และ **ขวา** (รายชื่อบุคคล)

### เพิ่มบุคคลใหม่

1. กดปุ่ม **"+ เพิ่มบุคคล"** (สีส้ม ด้านขวาบน)
2. **เลือกรูปใบหน้า** ของคนที่ต้องการ
   - ควรเป็นรูปที่เห็น **ใบหน้าชัดเจน ตรงๆ** ไม่บัง ไม่เบลอ
   - รูปมีคนเดียวหรือหลายคนก็ได้ โปรแกรมจะเลือกใบหน้าที่ชัดที่สุด
3. **กรอกชื่อ** ของบุคคล แล้วกด OK
4. บุคคลจะปรากฏเป็น **การ์ด** ในหน้าจอ

### จัดการบุคคล

แต่ละการ์ดบุคคลจะมีปุ่ม:

| ปุ่ม | หน้าที่ |
|------|--------|
| **แก้ไข** | เปลี่ยนชื่อบุคคล |
| **ลบ** | ลบบุคคลออกจากฐานข้อมูล (ต้องยืนยัน) |
| **+ เพิ่มรูป** | เพิ่มรูปอ้างอิงเพิ่มเติม (เพื่อให้ค้นหาแม่นยำขึ้น) |
| **จัดการรูปอ้างอิง** | เปิดหน้าจัดการรูปอ้างอิงทั้งหมด (เลือกรูปหลัก/ลบรูป) |
| **จัดกลุ่ม** | เลือกกลุ่มสำหรับบุคคลนี้ |

### เพิ่มรูปอ้างอิงหลายรูป (แนะนำ!)

การเพิ่มรูปอ้างอิงหลายรูปจะทำให้ **ค้นหาแม่นยำขึ้นมาก**:

1. กดปุ่ม **"+ เพิ่มรูป"** บนการ์ดของบุคคลนั้น
2. เลือกรูปใบหน้าเพิ่มเติม (คนละมุม, ใส่แว่น, ยิ้ม, ไม่ยิ้ม ฯลฯ)

**เคล็ดลับ:** เพิ่มรูปอ้างอิง **1-3 รูป** ที่มีมุมและสภาพแสงต่างกัน จะได้ผลลัพธ์ดีที่สุด

### จัดกลุ่มบุคคล

ฝั่งซ้ายจะมี **แถบกลุ่ม** สำหรับจัดหมวดหมู่บุคคล:

1. กดปุ่ม **"+ สร้างกลุ่ม"** แล้วพิมพ์ชื่อกลุ่ม
2. **ลากรูปบุคคล** จากฝั่งขวา ไปวางบนกลุ่มทางซ้ายได้เลย
3. หรือกดปุ่ม **"จัดกลุ่ม"** บนการ์ด แล้วเลือกกลุ่ม
4. คลิกที่ชื่อกลุ่ม เพื่อ filter แสดงเฉพาะบุคคลในกลุ่มนั้น
5. คลิกขวาที่กลุ่ม เพื่อเปลี่ยนชื่อหรือลบกลุ่ม

**เลือกหลายรายการ:** กดปุ่ม "เลือกหลายรายการ" แล้วติ๊กเลือกหลายคน จากนั้นลากเข้ากลุ่มพร้อมกันได้

### สลับมุมมอง

กดปุ่มมุมมองที่ด้านขวาของช่องค้นหา เพื่อสลับระหว่าง:

- **มุมมองการ์ด** — แสดงเป็นการ์ดรูปภาพ (ค่าเริ่มต้น)
- **มุมมองรายการ** — แสดงเป็นตาราง มีคอลัมน์ชื่อ, จำนวนรูปอ้างอิง, กลุ่ม

## การจัดเรียงไฟล์อัตโนมัติ

ไม่ว่าจะใช้วิธีสแกนก่อนตั้งชื่อ หรือค้นหาจากรายชื่อ โปรแกรมจะ **คัดลอก** (ไม่ใช่ย้าย) รูปที่ตรงกัน:

```
📁 งานบวชน้องเอ/
  ├── 📁 สมชาย/              ← โปรแกรมสร้างให้
  │   ├── 🖼 IMG_0003.jpg     ← คัดลอกมาให้
  │   ├── 🖼 IMG_0015.jpg
  │   └── 🖼 IMG_0042.jpg
  ├── 📁 สมหญิง/
  │   ├── 🖼 IMG_0005.jpg
  │   └── 🖼 IMG_0022.jpg
  ├── 🖼 IMG_0001.jpg          ← ไฟล์ต้นฉบับยังอยู่ครบ
  ├── 🖼 IMG_0002.jpg
  └── ...
```

> **รูปต้นฉบับไม่ถูกย้ายหรือลบ** — โปรแกรมแค่คัดลอกเข้าโฟลเดอร์ย่อย ปลอดภัย 100%

## เคล็ดลับเพื่อผลลัพธ์ที่ดีที่สุด

### การเลือกรูปอ้างอิง

- ใช้รูปที่เห็น **ใบหน้าชัดเจน ไม่เบลอ ไม่มืด**
- เพิ่ม **1-3 รูป** ต่อคน จากหลายมุมและสภาพแสง
- หลีกเลี่ยงรูปที่ใส่แว่นกันแดดหรือหน้ากาก

### การปรับค่าความคล้าย

- เริ่มจากค่า **0.45** (ค่าเริ่มต้น) ก่อน
- ถ้าได้รูปน้อยเกินไป → **ลดค่าลง** (เช่น 0.35)
- ถ้าได้รูปผิดคนมาปน → **เพิ่มค่าขึ้น** (เช่น 0.55)

### ประสิทธิภาพ

- ลดขนาดรูปสูงสุดใน Settings จาก 1280 เป็น 640 ถ้าต้องการความเร็ว
- เพิ่มจำนวนเธรดถ้าเครื่องมี RAM เหลือเยอะ
- **สแกนโฟลเดอร์ทำครั้งเดียว** — เพิ่มบุคคลใหม่แล้วค้นหาได้เลย ไม่ต้องสแกนซ้ำ

## คำถามที่พบบ่อย

**Q: รูปต้นฉบับจะถูกลบหรือเปลี่ยนแปลงไหม?**

> A: ไม่ครับ โปรแกรมแค่ **คัดลอก** รูปที่ตรงกันไปเก็บในโฟลเดอร์ย่อย ไฟล์ต้นฉบับยังอยู่ครบถ้วน

**Q: ต้องต่ออินเทอร์เน็ตไหม?**

> A: ไม่ต้องครับ โปรแกรมทำงาน **Offline ทั้งหมด** โมเดล AI อยู่ในเครื่อง รูปภาพไม่ถูกอัปโหลดไปไหน

**Q: รองรับรูปกี่รูป?**

> A: ไม่จำกัดครับ ทดสอบกับหลายพันรูปได้สบาย ยิ่งเครื่องสเปกดียิ่งเร็ว

**Q: เพิ่มโฟลเดอร์กิจกรรมใหม่ได้ไหม?**

> A: ได้ครับ แค่เพิ่มโฟลเดอร์ย่อยในโฟลเดอร์หลัก แล้วกดปุ่ม **รีเฟรช** ที่หัวตาราง

**Q: ถ้าเพิ่มรูปใหม่ในโฟลเดอร์ที่สแกนแล้ว ต้องทำยังไง?**

> A: กดปุ่ม **รีเฟรช** ที่หัวตาราง แล้วสแกนโฟลเดอร์นั้นใหม่

**Q: ค้นหาแล้วไม่เจอรูปของคนที่ต้องการเลย?**

> A: ลองเพิ่มรูปอ้างอิงหลายรูปที่หลากหลายมุม หรือลดค่าความคล้ายลง

**Q: เพิ่มบุคคลใหม่แล้ว ต้องสแกนโฟลเดอร์ใหม่อีกไหม?**

> A: ไม่ต้องครับ เพิ่มบุคคลใหม่แล้วค้นหาได้เลย ระบบจะเทียบกับข้อมูลใบหน้าที่สแกนไว้แล้ว

**Q: ข้อมูลถูกเก็บที่ไหน?**

> A:
>
> - **macOS:** `~/Library/Application Support/PuriPhotoSearch/`
> - **Windows:** `%APPDATA%\PuriPhotoSearch\`

## สรุปวิธีใช้แบบรวดเร็ว

**วิธีที่ 1 — สแกนก่อนตั้งชื่อทีหลัง (ง่ายที่สุด)**

```
เลือกโฟลเดอร์ → กดสแกน → ตั้งชื่อ → กดดำเนินการ → เสร็จ!
```

**วิธีที่ 2 — ค้นหาจากรายชื่อ**

```
เพิ่มบุคคล (เมนูรายชื่อ) → เลือกโฟลเดอร์ → ติ๊กเลือกบุคคล → กดดำเนินการ → เสร็จ!
```
